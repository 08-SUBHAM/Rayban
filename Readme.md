# Rayban Landing Page

A modern, interactive landing page for Rayban built with Shery.js

## Features
- Smooth animations and transitions
- Responsive design
- Interactive elements powered by Shery.js

## Credits
Special thanks to [Sheryians Coding School](https://www.youtube.com/@sheryians) YouTube channel for the inspiration and guidance in creating this project.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- [Shery.js](https://www.npmjs.com/package/sheryjs) - For creating smooth animations and effects



## Setup
1. Clone the repository
2. Open `Index.html` in your browser
3. No build process required - it works directly in the browser

## Deployment
This project is deployed using Vercel.
